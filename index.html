JavaScript Functions Explaination

Functions are one of the most important concepts in JavaScript. A function always takes some type of logic. To create a function, you'll need the key word "function", a prefered name for your function, followed by a parenthesis and also a curly braces. Functions can take in arguments or parameters.

To call a function for it to be executed, you have to re-type the name of the function followed by a parenthesis & enter

To pass a value to a function, you'll have to again go the same way of creating a function but you'll now pass variables in the parenthesis that is after the function's name and the variables should be seperated by a comma. Then call the function and place two values in the parenthesis and enter.

If you dont pass a value to the variable, JavaScript will print "undefined" for that variable.

RETURN- When you use the return key word, it takes whatever that value is that you're returning and it puts it inside the variable of that function. Another interesting thing about ruturn keyword is that it makes sure no code runs after it. console.log is actually a function on its own.
